#Musical Game of Life
A musical exploration of Conway's Game of Life.  Please view in 'Desktop' Chrome for now.

http://cellsounds.herokuapp.com/

You'll need node and npm to run locally.

    npm install -g brunch

    cd gameoflife/

    npm install

    bower install

    brunch watch --server
